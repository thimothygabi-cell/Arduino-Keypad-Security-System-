This project uses:
- Arduino Uno
- 4x4 Keypad
- Servo Motor
- Passive Buzzer

## Features
- Password protected door lock
- Servo unlock on correct PIN
- Buzzer alert on wrong PIN
- D key to delete digit
- Auto verification

## Password
Default password: 9250

## How to Use
1. Enter 4-digit PIN
2. Correct PIN unlocks servo
3. Wrong PIN triggers buzzer
